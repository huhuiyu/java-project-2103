# 大三开发环境项目

- [jdk下载地址](https://www.oracle.com/cn/java/technologies/downloads/)
- 打开一个可以执行java指令的终端的方法
  - 按`Ctr + ~`打开终端窗口
  - 输入`cd .\script\`切换到脚本目录
  - 输入`.\java_cmd.ps1`启动新终端
  - 输入`java -version`测试使用的java版本
  - 也可以直接右键点击`script`文件夹,选择`在集成终端中打开`, 然后执行步骤3和4
